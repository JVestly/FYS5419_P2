Repository for project 2 in FYS5419 on using Quantum Optimization techniques for the Traveling Salesman problem.

The **Code** folder contains
-*utils.py* with all classes and functions needed for testing.
-*tests.ipynb* contains all code for classical testing.
-*tests_qc_valid_hardware.ipynb* contains all codes for testing on IBM backend.

The **Figures** Folder contains all relevant figures from the experimentation.

To reproduce the experiments, clone the GitHub repository and execute the provided scripts. The notebooks are already ran, but you might run again to replicate the results. Note that some tests take a few hours to run.
